# Learning Data With Me

`LearningJourney` is a notebook-based repository for practicing Python and data
learning exercises. It currently contains day-2 class notes and task work in
Jupyter Notebook format.

## Notebooks

- `Class_Day2.ipynb` - classroom examples covering Python basics such as
  variables and simple statements
- `Task_Day2.ipynb` - practice exercises focused on list operations and related
  beginner tasks

## How To Open

Open the notebooks in Jupyter Notebook, JupyterLab, or VS Code:

```bash
jupyter notebook
```

Then choose either notebook from the file browser.

## Purpose

This repository is meant to capture progress over time while learning Python
through hands-on notebook exercises.
